[[コンテンツ紹介ページ（外部）]](https://vracademy.jp/introduction/3442-2/)

## 開発環境
Unity(2017.2.0f3), Oculus Rift, Oculus Touch

## 制作期間
2017年12月末 ~ 2018年2月末。
